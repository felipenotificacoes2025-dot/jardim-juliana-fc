# Jardim Juliana FC — Site institucional

Landing page institucional do **Jardim Juliana Futebol Clube**, bairro Jardim Juliana, zona leste de Ribeirão Preto - SP.

> *"Mais que um time, uma família."*

## Sobre

Site estático (HTML, CSS e JavaScript puro), responsivo e mobile-first, contando a história do bairro e do clube (fundado em 2015), com galeria de fotos da comunidade e da torcida **Febre Azul**.

## Estrutura

```
index.html      → página única (hero, história, clube, Febre Azul, galeria, CTA, rodapé)
styles.css      → estilos (paleta azul royal + celeste + dourado, baseada no escudo)
script.js       → menu, animações no scroll, galeria dinâmica e lightbox
img/            → escudo e imagens das seções
img/galeria/    → fotos da galeria (01.jpg ...)
CNAME           → domínio personalizado (jardimjulianafc.com.br)
```

## Como visualizar localmente

Abra o `index.html` no navegador.
